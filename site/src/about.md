# 炊紙(kashikishi) について

炊紙(kashikishi) is a text editor that utilizes GPU to edit text in a 3D space.

炊紙(かしきし)はGPUを用いて三次元空間中でテキストを編集することができるテキストエディタです。

GitHub の [mitoma/kashiki2][] 上で開発されています。詳細については [README.md][] を見てください。
